# Crypto Demo
----

## Installation
~~~
$ git clone https://github.com/SourabhSTashildar/crypto.git
$ cd crypto
$ npm install
~~~

## Usage
### Step 1: Generate Public and Private keys
~~~
$ node index.js
~~~

### Step 2: Encrypt
~~~
$ node encrypt.JS 'MESSAGE TO ENCRYPT'
~~~

### Step 3: Decrypt
~~~
$ node decrypt.js
~~~
